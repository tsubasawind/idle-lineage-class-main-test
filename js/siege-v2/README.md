# 攻城 V2 基礎架構

V2 與現有攻城隔離，預設 `SiegeV2.enabled = false`，目前不取代正式入口。

- 程式與存檔只用穩定英文 ID，中文只負責顯示。
- 未定數值一律填 `null`，不可偷偷套用舊攻城數值。
- 新素材放在 `assets/siege-v2/castles/<castle-id>/`。
- 每個動畫單位使用自己的 `manifest.json`，日後補影格只更新清單。
- 守軍資料已預留等級規則、HP/MP、AC/MR、命中、傷害、魔攻、減傷、回血、攻速、抗性、AI、技能、掉落與動畫。
- 技能資料已預留種類、傷害類型、冷卻、骰值、等級/魔攻倍率與狀態效果。

主控台執行 `SiegeV2.foundationReport()` 可檢查資料引用。目前肯特僅為結構草案，正式數值、戰鬥轉接、UI、城主與守城尚未啟用。
